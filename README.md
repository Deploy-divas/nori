# 💻 Checkpoint 2 - Engenharia de Software | Frontend | NORI

Este projeto é uma aplicação web responsiva e dinâmica desenvolvida como parte da avaliação do Checkpoint 2 (Turma 1ESPI). O objetivo é aplicar conceitos de HTML5 semântico, estilização com CSS3 moderna e interatividade com JavaScript.

O tema escolhido pela equipe foi: **Restaurante Japonês**.

🌐 **Link do Site no ar (GitHub Pages):** https://deploy-divas.github.io/nori/

---

## 🚀 Como Rodar o Projeto Localmente

Siga o passo a passo abaixo para clonar o repositório e executar o projeto na sua máquina através do VS Code.

### 📋 Pré-requisitos
Antes de começar, você vai precisar ter instalado:
1. **Git** (para clonar o projeto)
2. **VS Code** (IDE)
3. Extensão **Live Server** no VS Code (altamente recomendada para ver as atualizações em tempo real)
